e-prot – Manuale Utente

 

 

Configurazione del sistema

 

Il sistema e-prot è installato su sistemi hardware e software aventi le caratteristiche sotto riportate.

Per la parte server:
L’applicazione è Open Source. Può funzionare sia su sistemi operativi tradizionali(Windows) sia Open Source(Linux).

Application Server:Apache, Tomcat o JBOSS. Linguaggio Java


Per la parte client :
Sistema personal computer (PC) di adeguata potenza dotato di processore IntelTM


Per quanto riguarda la base dati sono previste due diverse soluzioni : una open source (POSTGESQL) per realtà medio-piccole ed una con database ORACLE per realtà medio-grandi come nel caso del Ministero dell’Interno

Browser: Explorer, Netscape, Mozilla, ecc.
